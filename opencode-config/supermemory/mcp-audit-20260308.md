## Auto-Invocation Mechanisms

Three patterns found for LIVE MCPs:

1. **Skill-based**: Skills define explicit MCP tool calls
   - distill: mcp_distill_browse_tools, mcp_distill_run_tool
   - context7: mcp_context7_resolve-library-id, mcp_context7_query-docs

2. **Agent-based**: Agents reference MCP tools in prompts
   - librarian agent: calls context7 tools for documentation lookup

3. **Skill-orchestrator auto-recommendation**: Detects task keywords → recommends skill
   - Documentation keywords → context7 skill (lines 151-175 of skill-orchestrator-runtime.md)
