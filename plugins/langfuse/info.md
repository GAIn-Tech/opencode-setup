# opencode-plugin-langfuse

LLM observability, tracing, and cost tracking via Langfuse.

- **Package**: `opencode-plugin-langfuse@0.1.8`
- **Source**: npm registry
- **Purpose**: Traces all LLM calls, tracks token usage and costs, enables prompt versioning and A/B testing
- **Dashboard**: https://langfuse.com (requires account for full features)
