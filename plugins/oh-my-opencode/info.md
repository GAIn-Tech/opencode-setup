# oh-my-opencode

Multi-agent orchestration framework for OpenCode.

- **Package**: `oh-my-opencode@3.16.0`
- **Source**: https://github.com/Yeachan-Heo/oh-my-opencode
- **Version**: 3.16.0 (custom in-repo baseline)

## Features
- 8 named agents (sisyphus, oracle, atlas, metis, momus, hephaestus, librarian, prometheus)
- 29 specialized agent definitions
- 46 skills with SKILL.md prompts
- 22 slash commands
- Hook system for runtime context injection
- Background task management
- Session continuity via session_id

## Configuration
Configured via `oh-my-opencode.json` — agent model overrides, MCP toggles, hooks.
