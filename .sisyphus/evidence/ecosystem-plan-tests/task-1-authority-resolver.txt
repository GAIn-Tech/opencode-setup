## Precedence Chain Implemented
1. Environment variables (OPENCODE_AGENT_{NAME}_MODEL, OPENCODE_CATEGORY_{NAME}_MODEL)
2. Home config (~/.config/opencode/oh-my-opencode.json)
3. Repo config (opencode-config/oh-my-opencode.json)
4. Defaults (hardcoded fallbacks)

## Source Identifiers
- `env-override` - Environment variable
- `home-config` - Home directory config
- `repo-config` - Repository config
- `default` - Hardcoded default
- `not-found` - No configuration

## Provenance Tracking
Every resolution includes:
- `modelId` - The resolved model ID
- `provider` - The provider (e.g., 'openai', 'google')
- `source` - Where the value came from
- `provenance` - Detailed path (e.g., 'file:/path/to/config.json:agents.atlas.model')

## Backwards Compatibility
The `getTelemetryMaps()` function returns `CATEGORY_TO_MODEL` and `AGENT_TO_MODEL` maps in the format expected by existing consumers.
